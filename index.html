Selectores básicos

  1. Selector de elementos: selecciona elementos en función del nombre del elemento, como "p" para seleccionar todos los párrafos. Ejemplo:

  2. Selector de ID: selecciona un solo elemento basado en el atributo id, como "#parrafo-dos" para seleccionar el elemento con id="parrafo-dos". Ejemplo:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Obcaecati, quo. Aut veniam dolores repellendus alias sint deleniti repudiandae illo consequuntur eos sed in perferendis officiis libero, error quam? Doloremque, consequuntur.

  3. Selector de clase: selecciona elementos según el atributo de clase, como ".example" para seleccionar todos los elementos con class="example". Ejemplo:

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Voluptas facilis consequuntur itaque sint id illum. Doloribus vel perspiciatis quidem mollitia saepe amet unde, atque deleniti distinctio nemo vero in? Necessitatibus?

Selectores de atributos

En CSS, los selectores de atributos le permiten seleccionar elementos en función del valor de sus atributos. Hay varios tipos de selectores de atributos, que incluyen:

  • [atributo]: selecciona elementos que tienen el atributo especificado, independientemente de su valor.
  • [atributo=valor]: Selecciona elementos que tienen el atributo especificado con el valor exacto.
  • [atributo~=valor]: selecciona elementos que tienen el atributo especificado con un valor que contiene una palabra especificada.
  • [atributo^=valor]: selecciona elementos que tienen el atributo especificado con un valor que comienza con una cadena especificada.
  • [atributo$=valor]: selecciona elementos que tienen el atributo especificado con un valor que termina con una cadena especificada.
  • [atributo*=valor]: selecciona elementos que tienen el atributo especificado con un valor que contiene una cadena especificada.

Selector universal (*)

El selector universal en CSS está representado por un símbolo de asterisco (*) y coincide con cualquier elemento en la página HTML. Las reglas CSS aplicadas al selector universal se aplicarán a todos los elementos de la página. Por ejemplo, para establecer un margen y un relleno predeterminados para todos los elementos de la página:

* { margin: 0; padding: 0; }

El selector universal tambien puede ser contextualizado, es decir, para que se aplique a todos los elementos dentro de un selector. Ejemplo:

  • Primavera
  • Verano
  • Otoño
  • Invierno

Pseudoclases

Las pseudoclases de CSS son selectores especiales que le permiten seleccionar elementos en función de un determinado estado o propiedad que no es inherente al elemento. Estas son algunas de las pseudoclases más utilizadas:

  • :link: Coincide con enlaces no visitados
  • :visited: Coincide con los enlaces visitados
  • :hover: Coincide con un elemento cuando el usuario pasa el mouse sobre él
  • :active:Coincide con un elemento cuando se está activando, como cuando se hace clic en un botón
  • :focus: Coincide con un elemento que tiene foco, como un campo de formulario que está actualmente seleccionado
  • :target: Esta pseudoclase en CSS se usa para seleccionar y diseñar el objetivo de un identificador de fragmento en la URL. El identificador de fragmento es la parte de una URL que sigue al símbolo "#" y hace referencia a un elemento dentro de un documento. Al usar la :targetpseudoclase, puede aplicar estilos específicos a un elemento cuando es el objetivo de un identificador de fragmento.

Pseudoelementos

Los pseudoelementos CSS se utilizan para seleccionar y diseñar una parte específica de un elemento. Se crean utilizando dos puntos (::) seguidos de una palabra clave. Algunos pseudo-elementos comunes incluyen:

  • ::before - utilizado para insertar contenido antes del elemento seleccionado.
  • ::after - se utiliza para insertar contenido después del elemento seleccionado.
  • ::first-letter - se utiliza para seleccionar y diseñar la primera letra de un elemento.
  • ::first:line - se utiliza para seleccionar y diseñar la primera línea de un elemento.
  • ::selection - se utiliza para seleccionar y diseñar la parte de un elemento que selecciona el usuario.

Los pseudoelementos a menudo se usan para agregar efectos visuales a los elementos o para cambiar la visualización del contenido sin cambiar el marcado HTML real. Ejemplos:

El algoritmo de CSS

El algoritmo CSS (hojas de estilo en cascada) es un proceso utilizado por los navegadores web para determinar los estilos que se aplican a los elementos de una página web. El algoritmo CSS se utiliza para resolver conflictos entre diferentes estilos que se pueden aplicar a un solo elemento y para decidir qué estilos tienen prioridad en caso de estilos en conflicto. El algoritmo CSS funciona según la naturaleza en cascada de los estilos CSS, lo que significa que los estilos se pueden heredar de los elementos principales a los elementos secundarios, y que los estilos más específicos pueden anular los estilos más generales. Aquí hay un resumen de cómo funciona el algoritmo CSS:

1. El navegador recopila todos los estilos aplicables a un elemento, incluidos los estilos definidos en hojas de estilo externas, estilos definidos en el encabezado del documento y estilos definidos en línea en el HTML.
2. Luego, el navegador ordena los estilos según su origen e importancia, con los estilos de agente de usuario (estilos predeterminados aplicados por el navegador) que tienen la prioridad más baja y los estilos definidos directamente en un elemento que tiene la prioridad más alta.
3. Luego, el navegador procesa los estilos en el orden en que fueron ordenados, aplicando estilos de izquierda a derecha, y los estilos posteriores tienen prioridad sobre los estilos anteriores.
4. El navegador repite el proceso para cada elemento de la página, utilizando los estilos de los elementos principales para determinar los estilos predeterminados para los elementos secundarios.
5. Finalmente, el navegador representa la página, utilizando los estilos calculados para determinar el diseño y la apariencia de cada elemento.

Especificidad en CSS

La especificidad en CSS se refiere a la prioridad que tiene una regla CSS particular sobre otras reglas que se aplican al mismo elemento. El navegador utiliza la especificidad para determinar qué regla CSS debe tener prioridad cuando se aplican varias reglas al mismo elemento. La especificidad de CSS se calcula en función de cuatro factores: estilos en línea, ID, clases, atributos y pseudoclases, y elementos y pseudoelementos. A estos factores se les asigna un peso; los estilos en línea tienen el peso más alto y los elementos tienen el peso más bajo. Cuando se aplican dos reglas al mismo elemento, gana la regla con mayor especificidad.


La especificidad se calcula de la siguiente manera
Etiquetas y pseudo elementos ............. 0,0,0,1
Clases, atributos, pseudoelementos..... 0,0,1,0
Identificadores ...................................... 0,1,0,0
Estilos en linea ..................................... 1,0,0,0
!important .....................Rompe la especificidad

Herencia en CSS

En CSS, la herencia se refiere al mecanismo por el cual ciertas propiedades CSS se pasan de un elemento principal a sus elementos secundarios. Esto permite una forma más eficiente y concisa de diseñar un documento, ya que puede establecer estilos en un elemento principal y hacer que se apliquen automáticamente a sus elementos secundarios, en lugar de tener que configurar los estilos en cada elemento secundario individual.
Por ejemplo, si establece la propiedad de familia de fuentes en un elemento principal, cualquier texto dentro de ese elemento principal heredará la familia de fuentes. Sin embargo, si establece una propiedad de familia de fuentes diferente en un elemento secundario, esa propiedad anulará la familia de fuentes heredada del padre.
Es importante comprender que no todas las propiedades de CSS se heredan. Por ejemplo, las propiedades de ancho y alto no se heredan, ya que cada elemento tiene su propio ancho y alto. Puede encontrar una lista de propiedades heredadas en la especificación CSS.


En CSS, el inheritvalor se usa para especificar explícitamente que una propiedad debe heredar su valor de su elemento principal. Cuando una propiedad se establece en inherit, toma el mismo valor calculado que la propiedad correspondiente del elemento principal.
Por ejemplo, si tiene un elemento principal con un tamaño de fuente de 16 px y desea que un elemento secundario herede ese tamaño de fuente, puede establecer la propiedad de tamaño de fuente del elemento secundario en inherit pero si queremos evitarla podemos asignar otro valor o inicializar la propiedad en cuestion con el valor initial


p { font-size: 16px; } span { font-size: inherit; }
En este ejemplo, el spanelemento secundario tendrá un tamaño de fuente de 16 px, porque ha heredado el tamaño de fuente de su pelemento principal.

Reseteo y Normalizacion de estilos

El reseteo de CSS es un conjunto de reglas de CSS que establece los valores predeterminados para los elementos en una línea de base consistente en diferentes navegadores. El propósito de un reseteo de CSS es garantizar que todos los elementos tengan una apariencia y un comportamiento coherentes, independientemente del navegador que se utilice para verlos.
Un reseteo de CSS funciona configurando explícitamente estilos para elementos HTML comunes en valores neutrales y uniformes. Por ejemplo, un reseteo de CSS podría establecer el margen, el relleno y el tamaño de fuente de todos los elementos en cero, de modo que todos sean iguales de forma predeterminada. Esto garantiza que cualquier estilo que agregue a su propia hoja de estilo se aplicará de manera consistente, independientemente de los estilos predeterminados establecidos por diferentes navegadores.
