Estilos de fuentes y Textos

Font-Family

La propiedad se usa para especificar la fuente de un elemento. Determina qué fuente debe usarse para representar el contenido de texto del elemento. La font-familypropiedad puede tomar uno o más nombres de familias de fuentes, que generalmente se enumeran en orden de preferencia. Si la primera fuente no está disponible en la computadora del usuario, el navegador probará con la siguiente fuente de la lista, y así sucesivamente.
Ejemplo
p { font-family: Arial, Helvetica, sans-serif; }
En este ejemplo, los pelementos usarán la fuente Arial si está disponible en la computadora del usuario. De lo contrario, el navegador intentará usar la fuente Helvetica. Finalmente, si ninguna de esas fuentes está disponible, el navegador recurrirá a una fuente sans-serif genérica. Es importante tener en cuenta que los font-familyelementos secundarios heredan la propiedad, por lo que puede establecer la familia de fuentes en un elemento principal y todos sus elementos secundarios usarán la misma fuente de forma predeterminada.

Font display

La propiedad font-display en CSS se usa para controlar cómo se muestra una fuente web mientras se carga. La propiedad le permite especificar un comportamiento alternativo para la fuente en caso de que no esté disponible de inmediato.
La font-displaypropiedad toma uno de los siguientes valores:
auto: este es el valor predeterminado y significa que el navegador elegirá el mejor comportamiento en función de su configuración predeterminada. Esto generalmente significa que el navegador ocultará el texto hasta que se cargue la fuente, para evitar cualquier "destello de texto sin estilo" (FOUT).
block: este valor obliga al navegador a ocultar el texto hasta que se cargue la fuente, al igual que auto. Sin embargo, también asegura que el texto permanezca oculto durante el tiempo que la fuente tarde en cargarse. Esto puede ayudar a evitar cambios de diseño y mantener una experiencia de usuario uniforme.
swap: este valor le dice al navegador que use inmediatamente una fuente alternativa (generalmente una fuente del sistema) hasta que se cargue la fuente personalizada. Esto puede evitar FOUT y garantizar que el texto sea legible, pero es posible que la fuente alternativa no se vea tan bien como la fuente personalizada.
fallback: este valor es similar a swap, pero permite que el navegador use una fuente similar a la fuente personalizada, en lugar de una fuente alternativa genérica. Esto puede ayudar a mantener un diseño y una experiencia de usuario coherentes.
optional: este valor le dice al navegador que use la fuente personalizada si está disponible, pero que no espere a que se cargue. En su lugar, el navegador utilizará inmediatamente una fuente alternativa. Esto puede ayudar a acelerar los tiempos de carga de la página y reducir el riesgo de cambios de diseño.

Text align

Esta propiedad en CSS se usa para controlar la alineación horizontal del texto dentro de un elemento. Esta propiedad se puede aplicar a cualquier elemento que contenga texto, incluidos encabezados, párrafos, listas y más.
Algunos posibles valores de la propiedad text-align:
left: este valor alinea el texto con el borde izquierdo del elemento.
right: este valor alinea el texto con el borde derecho del elemento.
center: Este valor centra el texto horizontalmente dentro del elemento.
justify: este valor extiende el texto para que cada línea ocupe todo el ancho del elemento, excepto la última línea que está alineada a la izquierda.
