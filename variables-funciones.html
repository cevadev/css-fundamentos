Custom Properties

Las propiedades personalizadas, también conocidas como variables CSS, son una forma de definir y reutilizar valores en CSS. Se declaran utilizando el --prefijo seguido de un nombre y un valor. Por ejemplo:
:root { --main-color: #ff0000; }
En este ejemplo, estamos definiendo una propiedad personalizada denominada --main-colory dándole un valor de #ff0000(que es rojo). Una vez que se define una propiedad personalizada, se puede usar en cualquier lugar de la hoja de estilo haciendo referencia a ella mediante la var()función. Por ejemplo:
body { background-color: var(--main-color); }
Este código establece el color de fondo del bodyelemento en el valor de la --main-colorpropiedad personalizada. Si tuviéramos que cambiar el valor de --main-colormás adelante en la hoja de estilo, el color de fondo se actualizaría automáticamente.

Funciones en CSS

Funcion url()

La url()función en CSS se usa para especificar un recurso, como una imagen o un archivo de fuente, que debe usarse para una propiedad CSS en particular. La función toma un único argumento, que es la URL del recurso que se utilizará. Aquí hay un ejemplo de cómo url()se puede usar la función para establecer la imagen de fondo de un elemento:
.my-element { background-image: url('path/to/image.jpg'); }
En este ejemplo, la url()función se usa para especificar la ruta a un archivo de imagen que debe usarse como imagen de fondo para el .my-elementelemento. La url()función también se puede usar en otros lugares, como en la @font-faceregla para especificar la URL de un archivo de fuente que se usará en una página web.
Es importante tener en cuenta que la URL especificada en la url()función es relativa a la ubicación del archivo CSS, no al archivo HTML. Esto significa que si el archivo CSS y el archivo de imagen están en carpetas diferentes, la URL debe tener esto en cuenta. Por ejemplo, si el archivo de imagen está en una subcarpeta de la carpeta donde se encuentra el archivo CSS, la URL se vería así:
.my-element { background-image: url('images/image.jpg'); }

Funcion min() max()

En CSS, las funciones min()y max()se utilizan para especificar un valor que es el resultado de una comparación entre dos o más valores. Estas funciones se pueden usar en varias propiedades de CSS, como ancho, alto, tamaño de fuente, relleno, margen y más.
min(): Esta función se utiliza para especificar el valor mínimo entre dos o más valores. La sintaxis de la min()función es:
min(value1, value2, value3, ...)
La min()función devuelve el valor más pequeño de los valores proporcionados. Por ejemplo, si usa min(100px, 50px, 200px), la función devolverá 50px.
max(): Esta función se utiliza para especificar el valor máximo entre dos o más valores. La sintaxis de la max()función es:
max(value1, value2, value3, ...)
La max()función devuelve el valor más grande de los valores proporcionados. Por ejemplo, si usa max(100px, 50px, 200px), la función devolverá 200px.
Estas funciones se pueden usar con cualquier propiedad CSS que tome un valor, como ancho, alto, tamaño de fuente, relleno, margen y más. Por ejemplo, puede usar max-widthpara establecer un ancho máximo para un elemento o min-heightpara establecer una altura mínima para un elemento. Ejemplos
width: min(100px, 100%);
width: min(100px, 100%);
padding: max(10px, min(5px, 50%));

Funcion clamp()

La funcion clamp() una función matemática que se usa para establecer un valor entre un valor mínimo y máximo, con un valor preferido en el medio. La clamp()función toma tres argumentos:
clamp(minimum, preferred, maximum)
El primer argumento minimumes el valor mínimo que puede tomar la propiedad, el segundo argumento es el valor preferido y el tercer argumento maximum es el valor máximo que puede tomar la propiedad.
La clamp()función se usa comúnmente para establecer el tamaño de un elemento de manera que se adapte al tamaño de su contenedor o al tamaño de la ventana gráfica. Por ejemplo, si desea que un elemento tenga un ancho mínimo de 100 px, un ancho preferido del 50 % y un ancho máximo de 300 px, puede usar el siguiente código CSS:
width: clamp(100px, 50%, 300px);
Esto establecerá que el ancho del elemento sea el 50 % del ancho de su contenedor, siempre que esté entre 100 px y 300 px. Si el ancho del contenedor es inferior a 100 px, el ancho del elemento se establecerá en 100 px. Si el ancho del contenedor es mayor que 300 px, el ancho del elemento se establecerá en 300 px.
